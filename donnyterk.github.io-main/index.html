Primarily Python backend work — REST APIs, automation scripts, data pipelines, and AI/ML integrations. I also take on full-stack projects when the frontend is React or a similar modern framework.

How do you handle project communication? +

Daily or bi-weekly updates via Slack/email, async by default. I use GitHub for code and Notion or Jira for tracking. I adapt to your workflow easily.

Do you sign NDAs and contracts? +

Yes — always. I sign NDAs for any confidential work and provide a clear service agreement before starting. Protecting your IP is a priority.

A focused MVP API can be done in 1–2 weeks. Larger systems are scoped per project. I always provide a clear timeline upfront and stick to it.

Yes — I offer a 30-day bug-fix warranty on all delivered work. Ongoing maintenance retainers are available for long-term projects.

Rates depend on project scope and complexity. I offer both fixed-price quotes for well-defined projects and hourly rates for ongoing work. Reach out for a free estimate.
